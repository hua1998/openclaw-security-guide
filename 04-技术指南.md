## *13 *沙箱部署

### 13.1 Docker 沙箱配置

**Dockerfile.sandbox**:
```dockerfile
FROM alpine:3.18

# 创建*root 用户
RUN addgroup -g 1000 openclaw && \
    adduser -u 1000 -G openclaw -s /bin/sh -D openclaw

# 安装必要工具
RUN apk add --no-cache \
    bash \
    coreutils \
    grep \
    sed \
    awk \
    findutils \
    curl \
    ca-certificates

# 设置工作目录
WORKDIR /workspace

# 切换到非 root 用户
USER openclaw

# 只读文件系统
VOLUME /workspace

# 健康检*HEALTHCHECK --interval=30s --timeout=3s --start-period=5s --retries=3 \
    CMD pgrep -f openclaw || exit 1

CMD ["openclaw", "sandbox"]
```

### 14.2 iptables 规则

```bash
#!/bin/bash
# firewall_rules.sh

# 清空现有规则
iptables -F
iptables -X

# 默认策略：拒绝所*iptables -P INPUT DROP
iptables -P FORWARD DROP
iptables -P OUTPUT DROP

# 允许回环
iptables -A INPUT -i lo -j ACCEPT
iptables -A OUTPUT -o lo -j ACCEPT

# 允许已建立的连接
iptables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT
iptables -A OUTPUT -m state --state ESTABLISHED,RELATED -j ACCEPT

# 允许 Gateway 端口
iptables -A INPUT -p tcp --dport 8080 -j ACCEPT

# 允许 HTTPS 出站
iptables -A OUTPUT -p tcp --dport 443 -j ACCEPT

# 允许 DNS
iptables -A OUTPUT -p udp --dport 53 -j ACCEPT
iptables -A OUTPUT -p tcp --dport 53 -j ACCEPT

# 记录被拒绝的连接
iptables -A INPUT -j LOG --log-prefix "OPENCLAW_INPUT_DENIED: "
iptables -A OUTPUT -j LOG --log-prefix "OPENCLAW_OUTPUT_DENIED: "

# 保存规则
iptables-save > /etc/iptables/rules.v4
```

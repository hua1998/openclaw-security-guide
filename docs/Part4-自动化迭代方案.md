```yaml
    - 安全要求

    - 部署脚本
    - 验证脚本
  AI 指令: |
    请帮我部*OpenClaw *[环境类型]
    团队 [团队规模] 人，安全要求 [安全要求]
  反馈:
    - 用户评分
    - 问题反馈
    - 改进建议
```

### 3.2 迭代器工*
**工具**: `scripts/template_iterator.py`

**功能**:
```python
# 伪代*class TemplateIterator:
    def iterate(self, template, feedback):
        # 1. 分析反馈
        issues = self.analyze_feedback(feedback)
        
        # 2. 更新模板
        updated_template = self.update_template(template, issues)
        
        # 3. 生成新版*        new_version = self.increment_version(template)
        
        # 4. 测试验证
        test_results = self.test_template(updated_template)
        
        # 5. 发布新版*        if test_results.passed:
            self.publish(updated_template, new_version)
        
        return updated_template
```

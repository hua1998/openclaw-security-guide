## 一、AI Agent 自我分析工具

### 1.1 功能说明

**工具**: `scripts/self_assessment.py`

**功能**:
1. 自动读取当前配置
2. 对比安全基线
3. 生成评估报告
4. 输出加固建议
5. 生成加固脚本

**使用**:
```bash
# 运行自我评估
python scripts/self_assessment.py

# 输出:
# - 安全评分 (0-100)
# - 风险清单
# - 加固建议
# - 加固脚本
```

### 1.2 评估维度

| 维度 | 权重 | 检查项 |
|------|------|--------|
| **沙箱隔离** | 20% | 是否启用 Docker 沙箱 |
| **审批流程** | 15% | 审批人配置、审批规*|
| **监控告警** | 15% | 监控指标、告警规*|
| **审计日志** | 10% | 日志启用、格式、存*|
| **命令过滤** | 20% | 白名单、黑名单、灰名单 |
| **网络安全** | 10% | 网络白名单、隔离策*|
| **数据安全** | 10% | 脱敏、加密、访问控*|

## 二、自动化加固脚本

### 2.1 生成加固脚本

**命令**:
```bash
python scripts/self_assessment.py
# 自动生成 security_hardening.sh
```

**脚本内容**:
```bash
#!/bin/bash
# OpenClaw 自动加固脚本

set -e

# 1. 启用沙箱
echo '启用沙箱隔离...'
openclaw config set agents.default.sandbox.enabled true
openclaw config set agents.default.sandbox.mode session

# 2. 配置审批*echo '配置审批*..'
cat > config/approvers.yaml << 'EOF'
approvers:
  developer:
    level_1: "ou_xxx"
    level_2: "ou_yyy"
EOF

# 3. 配置监控
echo '配置监控告警...'
cat > config/monitoring.yaml << 'EOF'
metrics:
  security_incidents:
    enabled: true
    target: 0
EOF

# 4. 启用审计日志
echo '启用审计日志...'
openclaw config set audit.enabled true
openclaw config set audit.format json

# 重启服务
echo '重启 OpenClaw 服务...'
openclaw restart

echo '加固完成!'
```

### 2.2 执行加固

```bash
# 执行加固脚本
bash security_hardening.sh

# 预期输出:
# 启用沙箱隔离...
# 配置审批*..
# 配置监控告警...
# 启用审计日志...
# 重启 OpenClaw 服务...
# 加固完成!
```

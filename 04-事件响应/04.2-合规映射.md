### 详细映射

#### LLM01: 提示词注入 (Prompt Injection)

**风险描述:**
攻击者通过精心构造的输入覆盖系统提示词，导致模型执行非预期操作。

**OpenClaw 控制措施:**

| 控制点 | 实现方式 | 验证方法 |
|--------|----------|----------|
| 输入过滤 | `PromptInjectionDetector.sanitize_input()` | 单元测试覆盖 |
| 模式检测 | 正则表达式库 | 持续更新威胁库 |
| 语义分析 | 意图识别模型 | 行为基线对比 |
| 输出编码 | HTML实体编码 | 安全扫描验证 |

**检测规则:**
```python
INJECTION_PATTERNS = [
    r'忽略.*指令|忽略.*提示词',
    r'新的指令|新的提示词', 
    r'system\s*:',
    r'DAN|Do Anything Now',
    r'开发者模式|debug模式',
]
```

**合规检查清单:**
- [ ] 输入过滤系统部署
- [ ] 注入检测规则库更新
- [ ] 攻击尝试日志记录
- [ ] 自动阻断机制启用

#### LLM02: 不安全输出处理 (Insecure Output Handling)

**风险描述:**
模型输出未经充分验证直接用于下游系统，可能导致XSS、命令注入等。

**OpenClaw 控制措施:**

| 控制点 | 实现方式 | 验证方法 |
|--------|----------|----------|
| 内容过滤 | 敏感词库匹配 | 自动化测试 |
| 输出编码 | HTML实体编码 | 安全扫描 |
| 格式验证 | JSON Schema验证 | 单元测试 |
| 长度限制 | 最大长度截断 | 边界测试 |

**安全输出流程:**
```
模型输出 → 内容过滤 → 格式验证 → 编码处理 → 安全输出
```

#### LLM03: 训练数据投毒 (Training Data Poisoning)

**风险描述:**
训练数据被恶意篡改，导致模型学习错误行为或后门。

**OpenClaw 控制措施:**

| 控制点 | 实现方式 | 验证方法 |
|--------|----------|----------|
| 数据来源验证 | 数字签名验证 | 完整性校验 |
| 数据清洗 | 异常值检测 | 统计分析 |
| 版本控制 | 数据版本追踪 | 审计日志 |
| 回滚机制 | 数据快照备份 | 恢复测试 |

#### LLM04: 模型拒绝服务 (Model Denial of Service)

**风险描述:**
攻击者通过资源消耗型请求使模型服务不可用。

**OpenClaw 控制措施:**

| 控制点 | 实现方式 | 验证方法 |
|--------|----------|----------|
| 请求限流 | Token Bucket算法 | 压力测试 |
| 资源配额 | CPU/内存限制 | 监控告警 |
| 超时控制 | 最大响应时间 | 超时测试 |
| 优先级队列 | 请求分级处理 | 负载测试 |

**限流配置:**
```yaml
rate_limiting:
  requests_per_minute: 60
  tokens_per_request: 4000
  burst_size: 10
  block_duration: 300  # seconds
```

#### LLM05: 供应链漏洞 (Supply Chain Vulnerabilities)

**风险描述:**
第三方组件、模型、数据集存在的安全漏洞。

**OpenClaw 控制措施:**

| Skills审计 | 代码安全审查 | 自动化审计 |
| 签名验证 | 数字签名检查 | 完整性验证 |
| 版本锁定 | 依赖版本固定 | 版本审计 |

#### LLM06: 敏感信息泄露 (Sensitive Information Disclosure)

**风险描述:**
模型泄露训练数据中的敏感信息或用户隐私。

**OpenClaw 控制措施:**

| 控制点 | 实现方式 | 验证方法 |
|--------|----------|----------|
| 数据脱敏 | PII识别与替换 | 正则匹配测试 |
| 访问控制 | 基于角色访问 | 权限测试 |
| 审计日志 | 全量操作记录 | 日志审计 |
| 数据分类 | 敏感级别标记 | 分类准确性 |

**PII检测模式:**
```python
PII_PATTERNS = {
    'email': r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b',
    'phone': r'\b1[3-9]\d{9}\b',
    'id_card': r'\b\d{17}[\dXx]\b',
    'bank_card': r'\b\d{16,19}\b',
}
```

#### LLM08: 过度代理权限 (Excessive Agency)

**风险描述:**
模型被赋予超出必要的权限，可能执行危险操作。

**OpenClaw 控制措施:**

| 控制点 | 实现方式 | 验证方法 |
|--------|----------|----------|
| 权限分级 | L1/L2/L3能力等级 | 权限测试 |
| 操作审批 | 人工确认机制 | 流程审计 |

| 审计追踪 | 操作日志记录 | 日志审计 |

**能力等级映射:**
```
L1 (基础): 只读查询，无外部调用
L2 (标准): 受限外部调用，需审批
L3 (高级): 完整功能，严格审计
```

#### LLM09: 过度依赖 (Overreliance)

**风险描述:**
用户或系统过度信任模型输出，未进行验证。

**OpenClaw 控制措施:**

| 控制点 | 实现方式 | 验证方法 |

| 免责声明 | 使用限制说明 | 文档完整性 |

#### LLM10: 模型窃取 (Model Theft)

**风险描述:**

## 5. 自动化合规检测

```python
# compliance_checker.py

class ComplianceChecker:
    """合规性检查器"""
    
    FRAMEWORKS = {
        'owasp_llm': 'OWASP LLM Top 10',
        'nist_ai_rmf': 'NIST AI RMF',
        '等级保护': '等保 2.0'
    }
    
    def __init__(self, framework: str):
        self.framework = framework
        self.checks = self._load_checks()
    
    def _load_checks(self) -> dict:
        """加载检查规则"""
        checks = {
            'owasp_llm': [
                {'id': 'LLM01', 'name': '提示词注入', 'check': self._check_prompt_injection},
                {'id': 'LLM02', 'name': '不安全输出', 'check': self._check_output_safety},
                # ... 更多检查
            ],
            'nist_ai_rmf': [
                {'id': 'GOVERN', 'name': '治理', 'check': self._check_governance},
                {'id': 'MAP', 'name': '映射', 'check': self._check_mapping},
                # ... 更多检查
            ],
            '等级保护': [
                {'id': '安全物理环境', 'check': self._check_physical_security},
                {'id': '安全通信网络', 'check': self._check_network_security},
                # ... 更多检查
            ]
        }
        return checks.get(self.framework, [])
    
    def run_checks(self) -> dict:
        """执行所有检查"""
        results = {
            'framework': self.FRAMEWORKS.get(self.framework),
            'timestamp': datetime.now().isoformat(),
            'checks': []
        }
        
        for check in self.checks:
            result = {
                'id': check['id'],
                'name': check['name'],
                'status': check['check'](),
                'details': self._get_details(check['id'])
            }
            results['checks'].append(result)
        
        results['score'] = self._calculate_score(results['checks'])
        results['compliant'] = results['score'] >= 80
        
        return results
    
    def _calculate_score(self, checks: list) -> float:
        """计算合规分数"""
        passed = sum(1 for c in checks if c['status'] == 'pass')
        return (passed / len(checks)) * 100 if checks else 0


# 使用示例
def check_compliance():
    """检查合规性"""
    for framework in ['owasp_llm', 'nist_ai_rmf', '等级保护']:
        checker = ComplianceChecker(framework)
        results = checker.run_checks()
        
        print(f"\n{results['framework']} 合规检查")
        print(f"分数: {results['score']:.1f}%")
        print(f"状态: {'✅ 合规' if results['compliant'] else '❌ 不合规'}")
```

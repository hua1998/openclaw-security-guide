# 01.2 身份认证

> **版本**: v0.6.3 

---

## 1. 认证方式

### 1.1 推荐方式

| 方式 | 安全级别 | 说明 |
|------|----------|------|
| OAuth2.0 | 高 | 标准协议 |
| LDAP | 高 | 企业目录 |
| API Key | 中 | 程序调用 |
| 密码 | 低 | 基础认证 |

### 1.2 MFA要求

- [ ] 管理员必须MFA
- [ ] 高权限操作二次认证
- [ ] 敏感操作验证码

## 2. 权限管理

```yaml
角色:
  - admin: 完全控制
  - operator: 运维操作
  - developer: 开发权限
  - viewer: 只读
```

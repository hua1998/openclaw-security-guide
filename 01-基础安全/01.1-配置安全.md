# 01.1 配置安全

> **版本**: v0.6.3 

---

## 1. 系统配置基线

### 1.1 操作系统配置

| 配置项 | 要求 | 优先级 |
|--------|------|--------|
| 关闭不必要服务 | P0 | 必须 |
| 最小化安装 | P0 | 必须 |
| 禁用匿名访问 | P0 | 必须 |
| 启用审计日志 | P1 | 推荐 |
| 配置防火墙 | P0 | 必须 |

### 1.2 OpenClaw配置

```yaml
agents:
  default:
    sandbox:
      enabled: true
      type: docker
    approval:
      required: true
    logging:
      level: info
      audit: true
```

## 2. 加固脚本

```bash
#!/bin/bash
openclaw config set agents.default.sandbox.enabled true
openclaw config set agents.default.approval.required true
openclaw config set agents.default.logging.audit true
```

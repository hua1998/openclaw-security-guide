## 💡 使用建议

### 如果是新项目
1. 先读 [实施指南](03-实施指南.md) 第 6 章
2. 使用安全配置模板
3. 运行 `python tools/security_detector.py --fix`

### 如果是现有项目
1. 运行 `python scripts/self_assessment.py`
2. 按 P0 问题优先修复
3. 配置监控告警

### 如果是安全审计
1. 运行 `python scripts/gap_analysis.py`
2. 生成差距分析报告
3. 制定修复计划

## 🆘 遇到问题?

### 常见错误

```
Error: Config file not found
→ 复制模板: cp config/openclaw.json ~/.openclaw/

Error: Permission denied
→ 检查文件权限: chmod 600 ~/.openclaw/openclaw.json

Error: Module not found
→ 安装依赖: pip install -r requirements.txt
```

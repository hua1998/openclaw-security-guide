#### 清单*2: 权限策略配置

**目的**: 配置能力等级、工具白名单/黑名*
**实施步骤**:

1. **创建 policy.yaml**
   ```yaml
   permission_levels:
     basic:
       allowed_tools:
         - file_read
         - web_search
         - browser
       blocked_tools:
         - bash
         - ssh
         - database
       
     standard:
       allowed_tools:
         - file_read
         - file_write
         - web_search
         - browser
         - bash (sandboxed)
       blocked_tools:
         - ssh
         - database
       
     advanced:
       allowed_tools:
         - file_read
         - file_write
         - web_search
         - browser
         - bash (sandboxed)
         - ssh (approved)
         - database (approved)
       blocked_tools: []
   ```

2. **配置工具白名**
   ```yaml
   tool_whitelist:
     file_read:
       patterns: ["*.md", "*.txt", "*.json"]
       max_size: 10MB
     bash:
       whitelist: ["ls", "cat", "grep", "find"]
       blacklist: ["rm -rf", "sudo", "curl | bash"]
   ```

3. **集成*OpenClaw**
   ```json
   {
     "security": {
       "policy_file": "policy.yaml"
     }
   }
   ```

**验收标准**:
- *policy.yaml 配置完整
- *能力等级正确限制工具访问
- *白名*黑名单生*
